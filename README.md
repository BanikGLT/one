# Telegram Mini App - Roulette Game

Игра-рулетка для Telegram Mini Apps с барабаном, таймером и системой гифтов.

## Технологии

- React 19
- Vite
- TailwindCSS
- ESLint

## Разработка

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Деплой

Проект готов к деплою на любом облачном хостинге с поддержкой Node.js.
